# WICARA UI Icons

The current Kamus and Progress screens use code-native illustrations from
`WicaraIllustrationIcon`. Place final transparent PNG or WebP icon exports in
this folder when a raster illustration set is ready.

Recommended filenames:

- `dictionary.webp`
- `search.webp`
- `subject.webp`
- `predicate.webp`
- `object.webp`
- `adverb.webp`
- `progress.webp`
- `statistics.webp`

Export each icon at 256 x 256 px with a transparent background.
